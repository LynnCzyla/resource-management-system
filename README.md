# Resource Management Recommender System

A **Resource Management Recommender System** designed to help organizations match employees’ skills with project needs more accurately and efficiently.  
This system uses **Python**, **Tesseract OCR**, and **Natural Language Processing (NLP)** to analyze resumes, certificates, and extracting relevant skills and suggesting the most suitable employee for a project.

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (FastAPI)  
- **Database:** Supabase (PostgreSQL)  
- **Containerization:** Docker  

---

## Features

- User authentication and role management  
- CRUD operations for resources and employees  
- Real-time database interaction with Supabase
- Recommend employee suitable for a project requirements
